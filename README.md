# UPI-Fraud-Detection-Analysis
A machine learning and deep learning based fraud detection system designed to identify suspicious UPI transactions in real time. The project evaluates multiple classification algorithms including Logistic Regression, K-Nearest Neighbors, Support Vector Machine, Naive Bayes, Decision Tree, Random Forest, and Convolutional Neural Network (CNN) using a UPI transaction dataset.
The system includes an interactive web interface where users can upload datasets, train models, and test transaction details to determine whether a transaction is legitimate or potentially fraudulent. The platform provides fraud risk assessment, confidence scores, and transaction analysis through an intuitive dashboard.
Key Features
Dataset Upload and Model Training
Real-Time UPI Transaction Analysis
Fraud Risk Scoring
Multi-Algorithm Performance Comparison
Responsive Web Dashboard
CNN-based High Accuracy Fraud Detection
